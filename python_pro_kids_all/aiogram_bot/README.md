# Quiz Kids com Aiogram

Bot educativo para Telegram feito com Aiogram. O projeto propõe um quiz simples para crianças com perguntas curtas, pontuação e comandos fáceis.

## Funcionalidades
- `/start`
- `/quiz`
- `/pontos`
- Respostas por mensagem

## Como executar
1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
2. Crie um bot no Telegram com o BotFather.
3. Defina o token em `TELEGRAM_BOT_TOKEN`.
4. Execute:
   ```bash
   python main.py
   ```

## Objetivo pedagógico
Estimular leitura, memória, atenção e participação em ambiente de chat.
