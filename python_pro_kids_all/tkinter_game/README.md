# Jogo Matemático Infantil com Tkinter

Projeto com interface gráfica voltado para crianças. O jogador responde continhas simples de soma e subtração e acompanha sua pontuação.

## Funcionalidades
- Interface gráfica amigável
- Perguntas aleatórias
- Soma e subtração
- Pontuação e rodadas
- Feedback de acerto e erro

## Como executar
```bash
python main.py
```

## Objetivo pedagógico
Desenvolver raciocínio lógico, atenção e familiaridade com interfaces gráficas em Python.
