# Projetos Python Pro para Crianças

Este pacote contém 4 projetos educativos, cada um usando uma tecnologia pedida na atividade:

1. **Discord.py** — bot de quiz infantil para Discord
2. **Aiogram** — bot de quiz infantil para Telegram
3. **Tkinter** — jogo matemático com interface gráfica
4. **Flask** — quiz infantil em página web

Cada pasta contém:
- `main.py`
- `requirements.txt`
- `README.md`

## Estrutura
- `discord_bot/`
- `aiogram_bot/`
- `tkinter_game/`
