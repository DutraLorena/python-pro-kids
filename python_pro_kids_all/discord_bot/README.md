# Bot Kids Quiz com Discord.py

Projeto educativo voltado para crianças. O bot envia perguntas simples de matemática e conhecimentos básicos, registra pontuação e incentiva a participação.

## Funcionalidades
- Comando `!ola`
- Comando `!pergunta`
- Comando `!responder <resposta>`
- Comando `!pontos`

## Como executar
1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
2. Crie um bot no Discord Developer Portal.
3. Defina o token na variável de ambiente `DISCORD_BOT_TOKEN`.
4. Execute:
   ```bash
   python main.py
   ```

## Objetivo pedagógico
Trabalhar leitura, atenção, raciocínio lógico e interação por comandos.
