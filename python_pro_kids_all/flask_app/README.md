# Quiz Infantil com Flask

Aplicação web simples voltada para crianças. A página mostra uma pergunta por vez, recebe a resposta e soma pontos.

## Funcionalidades
- Página web com quiz
- Pontuação na sessão
- Perguntas aleatórias
- Feedback imediato

## Como executar
1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
2. Execute:
   ```bash
   python main.py
   ```
3. Abra o navegador em `http://127.0.0.1:5000`

## Objetivo pedagógico
Mostrar como Python pode ser usado para criar aplicações web educativas.
