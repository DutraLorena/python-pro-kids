# 🎮 Python Pro Kids

Projeto educativo desenvolvido em Python com foco em crianças.

## 🚀 Tecnologias utilizadas
- Tkinter (interface gráfica)
- Flask (aplicação web)
- Discord.py (bot para Discord)
- Aiogram (bot para Telegram)

## 🎯 Objetivo
Ensinar lógica de programação de forma divertida e interativa.

## ▶️ Como executar
Cada pasta possui seu próprio projeto e instruções.
